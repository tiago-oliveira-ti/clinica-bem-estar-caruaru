# 🌿 Clínica Bem-Estar Caruaru

Site institucional fictício desenvolvido como projeto prático de **HTML e CSS**, simulando o site de uma clínica médica multidisciplinar.

---

## 📄 Páginas

| Página | Descrição |
|---|---|
| `index.html` | Página principal com apresentação e especialidades |
| `sobre.html` | História, valores, estrutura e equipe da clínica |
| `horarios.html` | Tabela de horários de atendimento por especialidade |
| `contato.html` | Informações de contato, mapa e formulário |

---

## 🚀 Tecnologias Utilizadas

- **HTML5** — estrutura semântica das páginas
- **CSS3** — estilização com variáveis, Grid Layout e Flexbox
- **Google Fonts** — tipografias Playfair Display e Lato
- **CSS externo** — separação de responsabilidades com arquivo `css/style.css`

---

## 📁 Estrutura do Projeto

```
clinica-bem-estar-caruaru/
├── index.html
├── sobre.html
├── horarios.html
├── contato.html
├── README.md
└── css/
    └── style.css
```

---

## ✨ Funcionalidades

- Layout com menu lateral fixo e navegação entre páginas
- Header com imagem de capa e sobreposição gradiente
- Cards de especialidades médicas
- Tabela de horários com tags coloridas (aberto/fechado)
- Formulário de contato com validação em JavaScript
- Mapa de localização incorporado (Google Maps)
- Paleta de cores consistente via variáveis CSS
